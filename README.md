# 🚀 Aurora Siger — Sistema Inteligente de Gerenciamento da Colônia Marciana

Sistema computacional desenvolvido em **Python** para gerenciamento inteligente e autônomo da colônia marciana **Aurora Siger**.

O projeto simula o funcionamento de uma base em Marte utilizando conceitos fundamentais de programação, lógica computacional e modelagem de dados, permitindo o controle inteligente de recursos essenciais para a sobrevivência da colônia.


## 🌌 Sobre o Projeto

O **Aurora Siger** foi criado com o objetivo de representar uma colônia marciana capaz de operar de maneira:

- autônoma;
- eficiente;
- segura;
- sustentável.

O sistema monitora dados internos da base e executa decisões automáticas a partir de regras operacionais previamente definidas.

Além disso, o projeto aplica conceitos matemáticos para análise energética e previsão de consumo utilizando regressão linear.



## 🧠 Funcionalidades Implementadas


### ✅ Organização Inteligente de Dados

A estrutura do sistema foi desenvolvida utilizando:

- dicionários;
- listas;
- estruturas hierárquicas;
- modularização de informações.


### ⚡ Gerenciamento Energético

O sistema realiza o monitoramento automático de:

- geração solar;
- geração eólica;
- armazenamento em baterias;
- consumo total da colônia.

Com isso, a base consegue identificar situações críticas e agir preventivamente para evitar falhas energéticas.


### 🤖 Tomada de Decisão Automatizada

O projeto utiliza regras lógicas para executar ações automáticas, como:

- ativação de protocolos de economia;
- redistribuição de energia;
- gerenciamento de módulos prioritários;
- controle operacional da colônia.



### 📈 Previsão Matemática

Foi implementado um modelo simples de **regressão linear** para prever tendências de consumo energético da colônia.

Esse recurso permite:

- antecipar sobrecargas;
- otimizar recursos;
- aumentar a eficiência operacional.


## 📥 Entrada e Saída de Dados

O sistema recebe informações simuladas de sensores ambientais e energéticos da colônia, incluindo:

- velocidade do vento;
- horário operacional;
- geração solar;
- nível das baterias;
- consumo energético dos módulos.

Esses dados são processados automaticamente para gerar:

- diagnósticos operacionais;
- previsões energéticas;
- alertas de risco;
- decisões automatizadas;
- relatórios do estado da colônia.

## 🔄 Lógica de Decisão Automatizada

O sistema executa decisões automáticas com base no balanço energético da colônia.

| Condição | Ação |
|---|---|
| Balanço < 0 e bateria ≤ 20% | Emergência total — desligar módulos não essenciais |
| Balanço < 0 e bateria ≤ 40% | Reduzir habitats e suspender laboratório |
| Balanço ≥ 0 e bateria ≤ 40% | Carregar baterias automaticamente |
| Excedente > 15 kWh | Sugerir armazenamento energético |
| Dentro dos parâmetros | Operação normal |
